# Добро пожаловать в проект MindMap
Проект выполнен студентами ВУЗа ВШЭ, проходящих обучение на майноре "UX-Дизайн"

Данный репозиторий – это [Expo](https://expo.dev) проект, сделанный [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Для начала

1. Установите необходимые утилиты. В частности [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

2. Установите зависимости

   ```bash
   npm install
   expo install @expo-google-fonts/inter
   ```

3. Запустите приложение

   ```bash
    npx expo start
   ```

В консоли вы найдете разные настройки приложения: 

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), песочница для Expo, имеет свои ограничения, но взамен очень удобна
